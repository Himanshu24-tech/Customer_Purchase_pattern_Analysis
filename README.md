# 🛒 Customer Purchase Pattern Analysis

## 📌 Project Overview

Customer Purchase Pattern Analysis is a data analytics project that analyzes customer purchasing behavior using transactional data from **3,900 customer records**. The primary objective of this project is to uncover valuable insights into customer spending habits, product preferences, subscription behavior, and purchasing patterns to support data-driven business decisions.

This project demonstrates an end-to-end data analytics workflow using **Python**, **PostgreSQL (SQL)**, and **Power BI**, covering data cleaning, exploratory data analysis, business analysis, and interactive dashboard development.

---

## 🎯 Objectives

* Analyze customer purchasing behavior and spending patterns.
* Identify high-value customer segments.
* Understand subscription and loyalty trends.
* Evaluate the impact of discounts and promotions.
* Discover top-performing products and categories.
* Generate actionable business insights through visualization.

## 🔄 Project Workflow

### 1. Data Cleaning & Preprocessing (Python)

## 📈 Key Insights

* Loyal customers contributed the highest revenue.
* Subscribers showed stronger purchasing behavior.
* Several products were highly dependent on discount strategies.
* Express shipping customers demonstrated higher average spending.
* Specific age groups generated significant business revenue.
